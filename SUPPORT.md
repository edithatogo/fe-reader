# Support

Use GitHub Issues for reproducible public bugs and GitHub Discussions or repository issues for documentation and packaging questions.

Do not attach private PDFs, document text, credentials, support bundles, crash dumps with private file paths, access tokens, private keys or exploit fixtures to public issues.

For security vulnerabilities, follow `SECURITY.md` and report privately.

For launch and installation status, check:

- <https://github.com/edithatogo/fe-reader/releases>
- `docs/stable-desktop-release.md`
- `packaging/registry-status.yaml`
