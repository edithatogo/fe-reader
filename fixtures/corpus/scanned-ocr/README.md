# Scanned OCR Fixtures

This folder is a Track AY placeholder for scanned PDF and searchable-PDF compatibility. It exists to keep OCR claims explicit and evidence-gated.

Current status:

- OCR is not enabled by default;
- searchable-PDF conversion is provider-gated and opt-in;
- no OCR fidelity claim is public until generated or redistributable fixtures and oracle results are accepted.

Accepted future fixtures must record source image provenance, language/script coverage, expected text, confidence thresholds and privacy handling.
