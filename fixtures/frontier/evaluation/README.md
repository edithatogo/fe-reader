# Frontier Evaluation Fixtures

Frontier intelligence evaluation fixtures must be synthetic, public-domain or
explicitly generated for testing. They must not contain private documents,
private user prompts, private support bundles, credentials, private file paths
or unpublished customer data.

The `frontier_intelligence_preview` gate may use these fixtures only for local,
offline evaluation. Any preview promotion needs a synthetic or public eval
report, privacy review, security review and resource-budget evidence.
