# Track S: Developer Ecosystem & SDK Spec

Track S owns stable public interfaces, examples, SDK docs, contract tests and plugin developer tooling.

## Deliverables

- public API versioning policy.
- SDK examples for Rust, CLI, MCP, Swift, Kotlin, C# and web postMessage.
- plugin devkit sample.
- contract test suite.
- deprecation policy.

## Dependencies

- C bindings/ABI.
- I/L automation and web surfaces.
- O security policy.
