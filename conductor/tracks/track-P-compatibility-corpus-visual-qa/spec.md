# Track P: Compatibility Corpus & Visual QA Spec

Track P owns fixtures, corpus governance, visual regression, differential testing and compatibility reports.

## Deliverables

- fixture manifest schema and validator.
- generated/public/private fixture policy.
- visual regression runner.
- differential renderer/converter reports.
- release compatibility report.

## Dependencies

- B rendering for actual image output.
- A text extraction for span golden tests.
- G standards/preflight for PDF/A/UA/X fixtures.
