# Fe Reader 0.1.0 Preview 1

Fe Reader 0.1.0 Preview 1 is an implementation checkpoint for the local-first PDF workflow platform. It is not a stable desktop launch.

## Included

- Headless Rust workspace with Wave 0 contract checks.
- CLI doctor and inspect smoke paths.
- Native macOS shell preview for local UI smoke testing.
- Release evidence scaffolding for SBOM, provenance, signing readiness and package matrix checks.
- Package manifest templates for GitHub Releases, Homebrew, Winget, Chocolatey, Scoop, Flatpak, Snap and AUR.

## Publication Policy

Desktop package publication remains blocked until signed artifacts, checksums, release evidence and explicit maintainer approval are present. Registry credentials must not be stored in this repository.
