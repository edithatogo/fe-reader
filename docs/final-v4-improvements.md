1. Security threat model and process isolation plan.
2. Compatibility corpus governance and visual regression testing.
3. Deterministic search/indexing without early ML dependency.
4. Text/font/i18n plan for CJK, RTL, vertical writing, ligatures and accessibility text.
5. Signed update and release-readiness contracts.
6. Enterprise policy and managed deployment plan.
7. Privacy-preserving diagnostics/observability.
8. Forms/JavaScript/XFA safety strategy.
9. Scanner/OCR ingestion roadmap.
10. Developer ecosystem and SDK versioning.

These additions should be treated as implementation requirements, not optional documentation, once each associated wave begins.
