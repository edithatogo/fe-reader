---
title: Rendering Performance Promotion
description: Evidence gates for rendering, GPU and performance claims.
---

The `rendering_performance_promotion` gate is disabled by default and does not block desktop stable launch.

Rendering promotion requires CPU-safe fallback, visual regression evidence, differential oracle evidence, performance budget reports and platform startup, memory, power and thermal summaries before public performance or GPU claims are expanded.

GPU paths remain optional and disabled until platform-specific evidence is complete.

See the full contract in [`docs/rendering-performance-promotion.md`](https://github.com/edithatogo/fe-reader/blob/main/docs/rendering-performance-promotion.md).
