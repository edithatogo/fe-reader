---
title: Ecosystem Integrations and Marketplace
description: Evidence gates for SDK, plugin, workflow-pack and marketplace expansion.
---

The `ecosystem_integrations_marketplace` gate is disabled by default and does not block desktop stable launch.

Publication requires compatibility snapshots, signed artifacts, SBOM, provenance, support and security links, maintainer approval and rollback plans. Plugin and integration mutations remain read-only or plan-only by default.

See [`docs/ecosystem-integrations-marketplace.md`](https://github.com/edithatogo/fe-reader/blob/main/docs/ecosystem-integrations-marketplace.md).
