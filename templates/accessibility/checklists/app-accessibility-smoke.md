# App Accessibility Smoke Checklist

- [ ] All primary actions reachable by keyboard.
- [ ] Focus indicator visible in default and high-contrast themes.
- [ ] Command palette can open document, search, annotate, inspect metadata and start workflows.
- [ ] Screen reader labels exist for toolbar, page list, document outline and review queues.
- [ ] Reduced-motion mode avoids non-essential animation.
- [ ] Touch/stylus controls have sufficient hit targets.
- [ ] Redaction and optimisation warnings are announced and visible.
