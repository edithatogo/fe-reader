# Maintainers

Fe Reader maintainership is organized by product and contract area.

## Roles

- Core Engine Maintainer
- Rendering Maintainer
- Platform Maintainer
- Security Maintainer
- Workflow Pack Maintainer
- Release Manager
- Compatibility Corpus Maintainer
- Documentation Maintainer

## Review Expectations

- Core contract changes require core and security review.
- Renderer, platform, MCP, plugin, COM, AppleScript, D-Bus, Android intent and iOS App Intent changes require adapter-boundary review.
- Public API changes require an API compatibility note and versioning decision.
- Release changes require evidence artifacts, minimal CI permissions and explicit timeouts.
